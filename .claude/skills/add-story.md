# Add Story

Add a new story to the "To Every Thing There Is a Season" Stories section on khurrambadar.com.

## Usage:
User tells a story or memory → transform into editorial prose with light humor and pizzazz.

## Steps:
1. Listen to the user's raw story/memory
2. Do deep dives on any people, places, or events mentioned
3. Write the story in Khurram's editorial voice: punchy, rhythmic, connecting dots, light humor (not fictional)
4. Create a Young Sherlock-style title (evocative, short, mysterious — like "The Briefcase", "The Kingmaker", "Behind the Microphone")
5. Assign to correct data-story category (manila/karachi/london/ksa/bahrain/dubai/singapore/markets/lesson)
6. Add to index.html stories section before the placeholder card
7. Commit and push

## Voice Rules:
- Write like someone who has SEEN things — not READ about them
- Short paragraphs. Punchy. Rhythmic. Never academic.
- Connect dots that others miss
- Very light humor — never fictional, never disrespectful
- Bold the key insight or punchline
- End with a line that makes the reader pause
